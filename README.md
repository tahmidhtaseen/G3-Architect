G3 Architects Landing Page

A modern, responsive landing page for an architecture firm built with HTML, CSS, and vanilla JavaScript.

🚀 Features

- Responsive design that works on desktop, tablet, and mobile
- Sticky navigation with smooth scrolling to sections
- Attractive hero banner with call-to-action
- Team gallery and feature highlights
- Statistics section with animated counters (design ready)
- Sponsor logo strip
- Login modal with JavaScript interactivity
- Clean, reusable CSS architecture

🛠️ Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Flexbox, custom properties, responsive units
